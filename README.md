# Student Grade Tracker

## Overview

Student Grade Tracker is a Java console-based application that helps users manage student marks and evaluate academic performance. It stores student records, calculates average, highest, and lowest scores, assigns grades, and generates a summary report.

## Features

* Add student names and marks
* Store data using ArrayList
* Calculate average, highest, and lowest scores
* Assign grades automatically
* Display a summary report
* Console-based user interface

## Technologies Used

* Java
* ArrayList
* Scanner Class
* Object-Oriented Programming (OOP)

## Requirements

* Java Development Kit (JDK 8 or above)
* Any Java IDE (Eclipse, IntelliJ IDEA, or VS Code)

## How to Run

1. Save the program as `StudentGradeTracker.java`.
2. Compile the program:

   ```
   javac StudentGradeTracker.java
   ```
3. Run the program:

   ```
   java StudentGradeTracker
   ```
4. Enter the number of students, their names, and marks.
5. View the generated student report and summary.
